■ゲームの概要

　白い箱を操作して敵を避けながらゴール地点を目指す２Ｄアクションゲームです。
　ゲームパッド推奨！


■注意

　このソフトは、フリー画像素材・フリー音楽（効果音）素材を使用してます。
　このソフト内で使用されている画像・音楽（効果音）を、このゲームを遊ぶ以外の用途には使用しないで下さい。


■動作環境

　Windows 10


■インストール方法

　アーカイブの中身をローカルディスク上の任意の場所にコピーして下さい。


■アンインストール方法

　レジストリなどは一切使っていません。
　ファイルを削除するだけでアンインストールできます。


■起動方法

　Hako2.exe を実行して下さい。


■操作方法

　左右キー       ... 左右に移動
　下キー         ... 床の際から下に降りる
　Z              ... ジャンプ（決定）
　X              ... 押しながら方向キーで低速移動（キャンセル）
　A              ... 押しながら方向キーで画面の移動・単に押すだけで画面を元に戻す
　S              ... クイックリスポーン
　C              ... リスポーン地点作成　★ＮＥＷ
　スペースキー   ... ポーズ
　エスケープキー ... 直ちにゲームを終了


■リスポーン地点の説明

　１フロア、３回だけリスポーン地点（中間地点）を作成できます。
　リスポーン地点作成後、死亡するかクイックリスポーンすると、リスポーン地点に戻されます。
　ポーズメニューのリスタートを選択すると、リスポーン地点がクリアされ、スタート地点に戻されます。

　●脱出不可能な場所・どうやってもリスポーンキルされる場所にリスポーン地点をセットしてしまった場合

　　例えば、地形的に先へ進めない場所にリスポーンとか、
　　敵を避けられない場所にリスポーンとか、そういった場合どうにもなりません。
　　あきらめてポーズメニューからリスタートして下さい。


■その他の説明

　マリオのＢダッシュのように１キャラ分の隙間は走り抜けられます。
　２キャラ分の隙間も走り抜けられます。

　床の端や１キャラ分の隙間から下に落ちたいとき、下ボタンを押すと落ちやすくなります。

　速度の速い敵は、壁にぶつかった時けっこうめり込みます。
　当たらないだろうと思ったところでも当たるかもしれません。
　１キャラ分以上めり込むことはないはずなので、壁の向こうにいれば大丈夫。多分。。


■制限

　リスポーン地点からプレイした場合はリプレイを保存できません。
　１フロアのプレイ時間が１時間を超えるとリプレイを保存できません。
　作成したリプレイデータは、同じバージョンでのみ再生可能です。


■蛇足

　プレイ中Ｒキーを押すと、直前の死亡リプレイを保存できます。（リプレイ有効時のみ）

　リスポーン地点を作成できる回数は、タイトルメニューの SETTING / RESPAWN COIN STOCK から、
　１フロア、０～１２回の間で変更できます。

　リスポーン地点からの再開時に星のエフェクトが邪魔だというとき、
　「クイックリスポーン」ボタンを長押しすると、星のエフェクトが出なくなります。

　ゴール地点は、水色に明滅しているそれっぽいやつです。
